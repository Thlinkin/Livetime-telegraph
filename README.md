# Livetime-telegraph
A telegraph sender modified so that messages are translated into readable English in live time. Formed with circuitry and code written in Arduino and C++. Includes translation, output, space and backspace compatibility, and additional qol features.

To run:
Follow report to observe wiring
Connect wired arduino breadboard to computer, choose correct port and preferred baud rate
Implement Arduino code and c++ code, match baud rates and port
The c++ IDE must be compatable with the external ImGui dependency.
Run c++ code; Inputs come in from Arduino breadboard, into arduino, into c++, and to a visual updating output.
